# ttb-routes-api-gateway-commons

This repo aims to centralize routes for Track to Buy backend integration.
All routes are RESTful endpoints registered in a Swagger.yml file and it deploys to directly to an AWS API Gateway.

## Prerequisites
All you have to have is a code editor which helps you write an .yml file in a easy way. VS Code or Google Antigravity are the most recommended ones in order to work/edit this repository.

### Plugins
In case you're using VS Code, we recommend that you install the [OpenAPI (Swagger) Editor](https://marketplace.visualstudio.com/items?itemName=42Crunch.vscode-openapi) plugin just because it helps you visualize the results of your updates without the need of accessing the Swagger Editor online.

## Installation
Besides the code editor and the recommed plugins, there are no other extra installation requirements.
